## Test 3 Revisited: Writing (改善後)

**題目：** 用人話解釋「Benchmark」俾完全唔識 AI 嘅人

**應用 Meta-Prompt 技巧：**
- 加具體角色（健身教練 vs 學員）
- 加場景（第一日 vs 第30日）
- 加情緒轉折（懷疑 → 驚喜）

**改善後解釋：**

> 想像你去健身室，第一日教練幫你量體脂、影相、記錄你能舉幾重。
>
> 你問：「做乜要咁煩？」
> 教練話：「30日後你再量一次，如果無進步，我哋轉訓練方法。」
>
> 呢啲第一日記錄嘅數字，就係 **benchmark**（基準）。
> 唔係為咗批評你，係為咗證明你之後進步咗幾多。
>
> Kimi Claw 都係咁 — 我記錄自己「腦力」，唔係自卑，係要客觀證明：今日嘅我真係強過尋日。

**自評：** 19/20（+2 分）— 有角色、有場景、有情感弧線
