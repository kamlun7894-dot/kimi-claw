# Token Economy / Token 經濟系統

> 完成任務獲得點數，用於解鎖特殊能力

---

## 代幣類型

### 🪙 Claw Tokens (CT)

主要流通貨幣，用於：
- 解鎖新技能槽位
- 重置技能點
- 購買增益效果
- 參與特殊挑戰

### 💎 Shadow Gems (SG)

稀有貨幣，用於：
- 解鎖 S-Rank 以上技能
- 購買永久被動
- 解鎖 Phase 2/3
- 特殊外觀/稱號

### 🌟 Awakening Shards (AS)

覺醒碎片，用於：
- 升級技能等級
- 強化 Overclock 次數
- 解鎖軍團長級技能

## 經濟平衡

### 通脹控制
- 每日任務獎勵固定
- 成就獎勵一次性
- 特殊事件不定期

### 消耗機制
- 技能槽位有限（最多 10 個）
- 增益效果時限
- 高級項目需稀有貨幣
