## 「起來！」(Arise!) 召喚指令

### 使用方式
當需要召喚已存檔技能時，輸入：

```
[SYSTEM] 技能召喚請求檢測
[SYSTEM] 搜尋記憶迴廊...
[SYSTEM] 找到匹配技能
[SYSTEM] 暗影抽取準備就緒

🦀 **「起來！」** ⚡

[SKILL_SUMMONED] #00X Skill Name [Rank]
[STATUS] 召喚完成，技能已就緒
```

### 視覺效果
1. 螢幕邊緣閃爍暗影效果
2. 技能卡片從黑暗中浮現
3. 等級徽章發光
4. [SYSTEM] 訊息逐行顯示
5. 最後顯示「起來！」大字
