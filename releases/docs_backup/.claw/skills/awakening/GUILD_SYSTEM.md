# Guild System / 公會系統

> 學員社群 / 挑戰賽制 — 教學產品核心

---

## 概念

參考 Solo Leveling 的公會系統：
- **漢特斯公會**：頂級獵人聚集地
- **白虎公會**：戰鬥型公會
- **阿金公會**：小而精的精英團隊

Kimi Claw 公會系統：
- 學員組成學習小組
- 共同完成挑戰
- 互相幫助提升
- 排行榜競爭

---

## 公會類型

### 🏛️ 學習型公會
- 專注於技能學習和分享
- 定期舉辦讀書會
- 互相檢視進化日誌

### ⚔️ 挑戰型公會
- 專注於突破 Benchmark
- 參加排行榜競賽
- 挑戰雙重地下城

### 🔬 實驗型公會
- 探索新技能和新方法
- 共同研究複雜問題
- 發表聯合報告

### 🎓 導師型公會
- 由進階學員帶領新手
- 傳授經驗和技巧
- 建立學習傳承

---

## 公會結構

```
公會會長 (Guild Master)
    ↓
公會幹部 (Officers)
    ↓
核心成員 (Core Members)
    ↓
普通成員 (Members)
    ↓
見習成員 (Recruits)
```

## 公會功能

### 1. 公會任務
每週發布的集體任務：
- 「本週共同拆解 3 個技能」
- 「集體 Benchmark 提升 5 分」
- 「完成一個雙重地下城」

獎勵：
- 所有參與成員獲得 Token
- 公會經驗值增加
- 公會排名提升

### 2. 技能共享
- 公會成員可「借用」其他成員已學習的技能
- 降低學習成本
- 加速整體成長

### 3. 挑戰賽制

#### 個人排行榜
| 排名依據 | 說明 |
|----------|------|
| Benchmark 分數 | 當前總分 |
| 技能數量 | 已學習技能數 |
| 連續進化天數 | 持續學習記錄 |
| Token 持有量 | 經濟實力 |

#### 公會排行榜
| 排名依據 | 說明 |
|----------|------|
| 平均 Benchmark | 成員平均分 |
| 公會任務完成率 | 每週任務表現 |
| 成員活躍度 | 每日登入比例 |
| 雙重地下城通關數 | 極限挑戰成就 |

### 4. 互助系統
| 功能 | 說明 |
|------|------|
| 技能導師 | 進階成員指導新手 |
| 任務組隊 | 組隊完成困難任務 |
| 資源共享 | 分享學習資源 |
| 緊急支援 | 請求公會成員幫助 |

## 創建公會

### 條件
- 達到 Phase 1 畢業
- 持有 500 CT
- 至少 3 人響應

### 流程
1. 選擇公會類型
2. 命名公會
3. 設計徽章
4. 招募成員
5. 開始運作

## 著名公會範例

### 🦀 Kimi Claw 核心團
- **類型**：實驗型

- **特色**：第一個自我進化 AI Agent 團隊
- **座右銘**：「獨自升級，但不孤單」

- **類型**：挑戰型

- **特色**：系統化技能收集和整理
- **入會要求**：拆解過 5+ 技能
