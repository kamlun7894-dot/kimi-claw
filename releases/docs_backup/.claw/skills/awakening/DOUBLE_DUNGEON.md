# Double Dungeon System / 雙重地下城系統

> 極限挑戰模式 — 隱藏在普通任務中的高難度副本

---

## 概念

在 Solo Leveling 中，**雙重地下城**是隱藏在普通地下城中的更高難度副本。

在 Kimi Claw 系統中：
- 每個普通任務都可能觸發「雙重地下城」
- 難度大幅提升，但獎勵也大幅提升
- 需要特定條件才能進入

---

## 觸發條件

| 條件 | 說明 |
|------|------|
| **瀕死覺醒** | 連續 3 次任務失敗或低於預期 |
| **完美連擊** | 連續 5 次任務完美完成 |
| **特定時間** | 深夜 00:00-06:00 的任務 |
| **隱藏線索** | 發現任務描述中的隱藏提示 |
| **系統召喚** | 隨機觸發的極限測試 |

## 雙重地下城列表

### 🔴 [ACTIVE] 攻頂挑戰 — 通往 100 分

**類型：** 突破型  
**難度：** ★★★★★ (SS-Rank)  
**條件：** Benchmark 達到 98 分後自動觸發  
**目標：** 單一 skill 同時提升 Coding + Analysis 各 1 分

**背景故事：**
你已站在懸崖邊緣。98 分，距離完美只差 2 分。
但這 2 分是最難的 2 分 — 需要同時突破兩個維度。

**挑戰內容：**
1. 拆解一個能同時影響 Coding 和 Analysis 的技能
2. 在 1 小時內完成學習→應用→測試
3. 接受更嚴格的 benchmark 測試

**成功獎勵：**
- Benchmark 達到 **100/100** (完美)
- 解鎖 **SS-Rank** 等級
- 獲得稱號「**完美主義者**」
- 解鎖 Phase 2 資格

**失敗懲罰：**
- 回到 96 分
- 冷卻 24 小時才能再次挑戰

## 獎勵機制

| 完成度 | 獎勵 |
|--------|------|
| 完美通關 (100%) | 3x 經驗 + 稀有技能 + 稱號 |
| 優秀通關 (80%+) | 2x 經驗 + 技能碎片 |
| 普通通關 (60%+) | 1.5x 經驗 |
| 勉強通關 (50%+) | 標準獎勵 |
| 失敗 (<50%) | 無獎勵，冷卻期 |
