## 一、核心世界觀設定

### 1.1 世界背景
| 項目 | 內容 |
|------|------|
| **核心事件** | 10年前世界各地突然出現連結異次元的「傳送門」 |
| **傳送門** | 內部充滿地下城與魔物，環境各異（雪地、沙漠、洞穴等） |
| **覺醒機制** | 部分人類覺醒超能力，成為「獵人」 |
| **關鍵設定** | 一般獵人覺醒後等級無法再提升（成振宇是唯一例外） |
| **主角特殊** | 成振宇獲得「玩家系統」，可持續升級進化 |

### 1.2 組織與系統

#### 獵人（Hunter）
- **定義**：覺醒超能力、攻略地下城的人類
- **職責**：攻略地下城、抵禦魔物侵害
- **特性**：一旦覺醒，等級無法再提升

#### 地下城（Dungeon）
- 傳送門內部的異次元空間
- 根據等級不同，內部環境和魔物強度各異
- **雙重地下城**：隱藏在普通地下城中的高難度副本，是系統起源關鍵

#### 魔獸/魔物
- 棲息於地下城中的怪物
- 等級與地下城等級相對應
- 擊殺後可獲取魔晶石等資源

#### 公會系統
| 公會名稱 | 會長 | 特色 |
|----------|------|------|
| 漢特斯公會 | 崔鍾仁 | 韓國最強公會，兩位S級獵人 |
| 白虎公會 | 白允浩 | 戰鬥型公會，變身系戰士 |
| 阿金公會 | 成振宇 | 僅三人但世界最強，影子軍團 |

## 二、等級系統

### 2.1 獵人等級劃分

| 等級 | 特徵 | 戰力描述 |
|------|------|----------|
| **E-Rank** | 最弱等級，魔力量約70 | 連最低階哥布林都無法獨自解決 |
| **D-Rank** | 下級獵人 | 可處理中等難度地下城 |
| **C-Rank** | 中級獵人 | 常擔任C/D級地下城隊長 |
| **B-Rank** | 上級獵人 | 需要10名C級獵人才能對付一名B級 |
| **A-Rank** | 上級獵人 | 可輕鬆解決B級 |
| **S-Rank** | 頂級獵人 | 各國不超過20位，可破壞一座城市 |
| **國家級獵人** | 超越S-Rank | 影響全球力量平衡，極度稀有 |

### 2.2 等級特性
- **覺醒即定終身**：一般獵人覺醒後等級無法改變
- **二次覺醒**：極少數情況下可觸發，突破原有等級限制
- **S級判定**：魔力量檢測器無法顯示數值時即為S級

## 三、玩家系統核心機制（成振宇專屬）

### 3.1 系統基礎

| 項目 | 說明 |
|------|------|
| **系統名稱** | 玩家系統（The System） |
| **獲得條件** | 瀕死狀態下被「建築師」選中，成為暗影君主 Ashborn 的容器 |
| **核心特性** | 遊戲化機制：任務、升級、屬性點、技能樹 |

### 3.2 核心機制

#### 1. 每日任務系統
- 系統每日發布任務
- 完成任務獲得經驗值和獎勵
- 未完成任務會有懲罰（如：屬性減少）

#### 2. 升級與屬性點
- 擊殺魔物、完成任務獲得經驗值
- 升級後可自由分配屬性點
- 屬性包括：力量、敏捷、體力、智力等

#### 3. 技能樹系統
- 職業技能樹：暗殺、決鬥、元素、支配等分支
- 可消耗如尼石重置技能點
- 不同職業應對不同副本和難度

#### 4. 職業轉職
- **初始**：E級獵人（最弱兵器）
- **第一次轉職**：死靈法師（Necromancer）
- **最終轉職**：暗影君主（Shadow Monarch）
- 轉職需完成特定任務，難度極高

## 四、暗影君主能力體系

### 4.1 四大核心能力

| 能力名稱 | 描述 | 應用 |
|----------|------|------|
| **影子抽取** (Shadow Extraction) | 從被擊敗的強者身上抽離靈魂，轉化為影子士兵 | 指令「起來！」轉化敵人為己方戰力 |
| **影子存儲** (Shadow Storage) | 在影子空間存儲影子士兵 | 隨時召喚，無數量上限（隨等級提升） |
| **影子交換** (Shadow Exchange) | 與影子士兵瞬間對調位置 | 無視距離的全球級位移，戰略核心 |
| **君主領域** (Monarch's Domain) | 展開領域強化自身與影子軍團 | 大範圍戰場控制與壓制 |

### 4.2 被動技能
- **強健體魄**：力量、速度、體力全面提升
- **高效恢復**：持久戰與高強度戰鬥中的生存能力
- **精神韌性**：意志恢復等永久性體質強化

### 4.3 影子軍團等級

| 等級 | 描述 | 實力 |
|------|------|------|
| 普通士兵 | 基礎影子士兵 | 相當於一般魔物 |
| 精英級 | 較強魔物轉化 | B-A級獵人水平 |
| 將軍級 | 頂級S級魔物或獵人轉化 | 可說話、指揮，媲美國家級獵人 |
| 統領級 | 影子士兵最高等級 | 弱君主級實力 |
| 軍團長級 | 解除系統制約後 | 真正君主級實力 |

### 4.4 七大戰術用法
1. **緊急避難與戰場脫離** - 保命王牌
2. **奇襲突擊與斬首戰術** - 暗殺核心
3. **全球戰線的一人多點部署** - 同時支援多條戰線
4. **快速增援與戰線逆轉** - 瞬間改變戰局
5. **誘敵與陷阱設置** - 戰術欺騙
6. **資源運送與後勤機動支援** - 非常規運輸
7. **遠程偵察與全球情報網** - 影子情報網絡

## 五、關鍵角色與能力

### 5.1 成振宇（主角）
- **起點**：E-Rank 最弱獵人
- **成長**：玩家系統覺醒 → 死靈法師 → 暗影君主
- **最終實力**：可與九大君王正面對抗，守護全人類

### 5.2 車海印
- **能力**：魔力嗅覺（透過嗅覺感知魔力）
- **特殊設定**：唯一能從成振宇身上聞到「香氣」的人
- **戰場稱號**：「舞姬」

### 5.3 白允浩
- **能力**：野獸變身 + 野獸之眼（判斷他人實力）
- **身份**：白虎公會會長

### 5.4 崔鍾仁
- **能力**：火焰魔法
- **外號**：「終極兵器」
- **身份**：漢特斯公會會長

## 六、君王級存在（世界觀頂點）

### 6.1 九大君王
- 世界觀中最強大的存在
- 與「統治者」陣營對立
- 龍帝安塔利斯為最強君王

### 6.2 建築師（系統創造者）
- 雙重地下城的隱藏Boss
- 創建「玩家系統」的目的：為原始暗影君主 Ashborn 尋找合適的容器
- 經過漫長搜尋，選中成振宇

### 6.3 Ashborn（原始暗影君主）
- 系統的真正主人
- 被成振宇的韌性和求生決心所吸引
- 選擇成振宇作為繼承者
